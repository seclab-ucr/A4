Whats New in Architecture Center?

Oracle VMware Cloud Solution

Oracle Cloud VMware Solution is a fully customer-controlled and customer-operated VMware environment provisioned on Oracle Cloud Infrastructure. It enables you to migrate VMware-virtualized workloads at a global scale, using existing practices for IT operations, and in a secure manner.

DNS for hybrid cloud

DNS resolution between Oracle Cloud and your on-premises network is a common requirement for enterprise applications. A hybrid DNS is an overlay deployment of DNS forwarders, configured to route DNS resolution requests between different private DNS namespaces.
